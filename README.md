
# 📦 Tarea 6 - Gestor de Inventario

Este proyecto corresponde a la Tarea 6 del curso de Front End, donde se desarrolla un gestor de inventario utilizando JavaScript.

## Contenidos desarrollados

- Filtrar productos con stock disponible usando `filter()`.
- Aplicar descuentos a los productos usando `map()`.
- Calcular el valor total del inventario usando `reduce()`.
- Encontrar el producto más caro usando `find()`.
- Simular el guardado del inventario mediante una `Promise`.
- Utilizar `.then()` y `.catch()` para manejar respuestas.
- Reescribir la solución utilizando `async/await`.

## Tecnologías utilizadas

- JavaScript
- Node.js
- Visual Studio Code

## Cómo ejecutar el proyecto

```bash
node tarea-6.js
